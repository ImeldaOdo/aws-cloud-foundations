## Diagnostic Commands Reference

### systemctl status nginx
Checks whether the NGINX service is running and reports recent errors.

### curl -I http://localhost
Confirms whether the web server is responding locally, independent of network access.

### ss -tuln
Displays listening ports to verify whether services are bound to expected ports.

## Linux Commands Used
- ls – list directory contents
- cd – change directory
- pwd – show current directory
- tail – view recent log entries
